1) Introduction

In a real program, you often need to read input (like a number from the keyboard) and then print output (like the computed result).

⭐ Remember

print() shows things.
input() reads things.

2) Theory

📚 Output vs Input

Input is data you get from the user (keyboard) or from a file.
Output is what you show or save (screen, file, etc.).

2.1 print() basics

print() outputs values to the screen. You can print variables and expressions.

2.2 input() basics

input(prompt) shows a prompt, waits for the user to type, and returns what the user typed.

⚠ Warning: input is always text

Even if the user types 25, input() returns "25" (a string). Convert it with int() or float() before doing math.

2.3 Formatting output

When output is clear, learning is easier. Use labels like "Age:" and "Total:".

You can combine strings and values using conversions. Later you will learn f-strings in detail, but for now you can use str() and concatenation.

3) Syntax

Python
print() syntax
# prints values to the screen
print("Hello")
print(x)
print(2 + 3)
Python
input() syntax
# reads a line from keyboard
name = input("Enter your name: ")
print("Hi, " + name)

Time complexity: O(1) for formatting and showing text • Space: O(1) extra (ignoring input size).

Alternative approach: store input then process it; don’t call input() many times in one expression.

4) Examples

4.1 Easy Example — greeting from input

Python
input → output
name = input("Enter your name: ")
print("Hello, " + name)

Output (example): If you type Ravi, output is Hello, Ravi.

Time: O(1) • Space: O(n) for storing name.

Alternative: if you already have name in a variable, just call print().

4.2 Medium Example — convert age to int

Python
string → number
age_text = input("Enter your age: ")
age = int(age_text)

age_next_year = age + 1
print("Next year you will be " + str(age_next_year))

Output (example): If input is 20, print Next year you will be 21.

Time: O(n) parsing digits • Space: O(1) extra.

Alternative: validate input before converting (later chapters add robust validation).

4.3 Advanced Example — simple calculator

Python
two inputs, one result
a_text = input("Enter first number: ")
b_text = input("Enter second number: ")

# convert text to integers
a = int(a_text)
b = int(b_text)

print("Sum = " + str(a + b))

Output: depends on user numbers.

Time: O(n) • Space: O(1) extra.

Alternative: use float() if you expect decimals.

6) Memory Visualization

input() stores the typed text into a variable. If you convert it, Python creates a new numeric value.

Variable Example value Type
age_text "20" str
age 20 int

🧠 Memory trick

“input” is “into text”. Convert it before math.

7) Dry Run

Python
dry run
score_text = "8"
score = int(score_text)
bonus = 2
print("Total: " + str(score + bonus))
Step Action Memory after
1 Store text score_text → "8" (str)
2 Convert score → 8 (int)
3 Add score + bonus → 10
4 Print Total: 10

Time: O(n) parsing • Space: O(1) extra.

Alternative: build the output using string conversion only once.

8) Built-in Functions

In this chapter you mainly use:

  • print()
  • input()
  • int(), float() for conversion
  • str() for formatting output
Python
conversion + print
n_text = "12"
n = int(n_text)
print("Double = " + str(2 * n))

Output: Double = 24

Time: O(n) • Space: O(1) extra.

Alternative approach: avoid repeated conversions by converting once.

9) Methods

Strings have methods. Since input() returns a string, you can use string methods to clean the text.

Python
strip() method
email = input("Enter email: ")
email = email.strip()
print("Saved: " + email)

Output: same email but without leading/trailing spaces.

Time: O(n) • Space: O(n) for new string.

Alternative: handle spaces using split() and joining (later).

15) Debugging Practice

Case 1: math on text

Python
buggy
a = input("Enter a number: ")
b = input("Enter a second number: ")
print(a + b)

Fix: convert before adding: print(int(a) + int(b)) (or float).

Case 2: label + number TypeError

Python
buggy
age = input("Age: ")
print("Next year will be " + age + 1)

Fix: convert and convert to string for printing: print("Next year will be " + str(int(age)+1)).

Case 3: invalid int conversion

Python
buggy
height = int(input("Enter height: "))
print(height)

Fix: if user types decimal, use float() instead.

16) Common Errors

  • TypeError: doing math with strings (forgot conversion).
  • ValueError: converting non-numeric text to int/float.
  • SyntaxError: missing quotes around prompt strings or labels.
  • NameError: using variable before assignment.

17) Best Practices

  • Convert input immediately after reading.
  • Use labels in print output.
  • Use str() when concatenating with strings.
  • Prefer storing input in variables before math.
  • Use strip() to remove extra spaces.
