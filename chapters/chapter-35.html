1) Introduction

NumPy is great for arrays. Pandas is great for tables.

In real work you often receive CSV files or data dumps. Pandas helps you load and analyze them quickly.

2) Theory

2.1 DataFrame = table in memory

A DataFrame has: rows (records) and columns (fields).

2.2 Selection: columns vs rows

  • Columns: pick by name.
  • Rows: pick by conditions or slicing.

2.3 Filtering with boolean masks

Filtering often looks like: df[df["age"] >= 18].

2.4 Grouping (split-apply-combine)

Grouping means: split the data into groups → apply a function → combine results.

⚠ Warning

Remember that boolean comparisons create a mask. You must use it inside df[mask].

3) Syntax

Python
read CSV + basic select
import pandas as pd

# Load CSV into a DataFrame
# Example file: people.csv with columns: name, age
# df = pd.read_csv("people.csv")

# For demo without file, we'll create a DataFrame

df = pd.DataFrame({
    "name": ["Asha", "Ravi", "Neha"],
    "age":  [20, 16, 22]
})

# Select one column
print(df["name"])

# Select rows by boolean mask
adults = df[df["age"] >= 18]
print(adults)

Time: depends on rows • Space: DataFrame stores the whole table.

Alternative: use pure CSV + lists, but Pandas is much easier for real tasks.

4.2 Medium Example — filter + sorting

Python
filter and sort
import pandas as pd

df = pd.DataFrame({
    "name": ["Asha", "Ravi", "Neha", "Sid"],
    "age":  [20, 16, 22, 18]
})

# Keep age >= 18
filtered = df[df["age"] >= 18]

# Sort by age descending
sorted_df = filtered.sort_values(by="age", ascending=False)

print(sorted_df)

Output: rows with age 18+ in descending age order.

4.3 Advanced Example — groupby summaries

Python
groupby mean
import pandas as pd

df = pd.DataFrame({
    "class": ["A", "A", "B", "B"],
    "score": [70, 90, 60, 80]
})

# Split by class, then compute mean score per class
summary = df.groupby("class")["score"].mean()

print(summary)

Output: mean score for each class.

Time: O(n log n) for many group/sort cases • Space: depends on groups.

15) Debugging Practice

Case 1: wrong boolean operator

bug
use and/or
# wrong
filtered = df[df["age"] > 18 and df["score"] > 80]

Fix: use & and | with parentheses.

Case 2: chained indexing confusion

bug
SettingWithCopy concept
adults = df[df["age"] >= 18]
adults["grade"] = "A"

Fix: use df.loc[mask, "grade"] = ... conceptually.

Case 3: missing column name

bug
typo column
df["agge"]  # typo

Fix: check df.columns and correct spelling.

16) Common Errors

  • KeyError: column name not found.
  • TypeError: comparing string to number.
  • ValueError: boolean mask not same length.
  • SettingWithCopyWarning: assignment on a slice (concept).

17) Best Practices

  • Inspect columns early (df.columns concept).
  • Use masks with parentheses and &/|.
  • Avoid chained indexing.
  • Convert dtypes before calculations.
  • Write small helper functions for repeated analysis steps.

22) Answer Key (Explain every answer)

Key explanations

  • DataFrame: a table structure with rows and columns.
  • Series: a single column in Pandas.
  • Boolean mask: a list/array of True/False used to filter rows.
  • Filtering: df[mask] keeps rows where mask is True.
  • Groupby: split data into groups, apply an aggregation, combine results.
  • Sorting: reorders rows based on values in column(s).
  • Why parentheses: boolean operations must be grouped correctly.
