1) Introduction

Python lists are flexible, but not optimized for heavy numeric work. NumPy arrays store data compactly and operate with fast code.

In this chapter you learn the core NumPy skills you need for data science.

2) Theory

2.1 Arrays instead of lists

A NumPy array is a grid of numbers. It has:

  • shape: how many rows/columns
  • dtype: the data type stored
  • size: total elements

2.2 Vectorization

Vectorized operations run on the whole array at once. Example: arr * 2 multiplies every element.

2.3 Broadcasting (concept)

Broadcasting lets NumPy combine arrays with different shapes when it is safe. For example, a scalar can be added to every element.

⚠ Warning

Shape mismatch causes errors. Always check arr.shape when things fail.

3) Syntax

Python
create arrays + basic ops
import numpy as np

arr = np.array([1, 2, 3])
print("arr:", arr)

arr2 = arr * 2
print("arr2:", arr2)

print("sum:", arr.sum())
print("mean:", arr.mean())

Time: O(n) • Space: O(n) for arrays.

Alternative: use pure Python loops, but they are usually slower.

4) Examples

4.1 Easy Example — indexing

Python
1D indexing
import numpy as np

a = np.array([10, 20, 30, 40])

print(a[0])   # first element
print(a[-1])  # last element

# slicing: start inclusive, end exclusive
print(a[1:3]) # elements at index 1 and 2

Output: 10, 40, and [20 30].

4.2 Medium Example — 2D indexing + slicing

Python
2D grid
import numpy as np

m = np.array([
    [1, 2, 3],
    [4, 5, 6]
])

print(m[0, 1])  # row 0, col 1
print(m[:, 2])  # all rows, col 2
print(m[0, :])  # row 0, all cols

Output: 2, [3 6], [1 2 3].

4.3 Advanced Example — vectorized transform

Python
vectorized math
import numpy as np

x = np.array([0, 1, 2, 3, 4], dtype=float)

# Example: compute y = x^2 + 2x + 1
# This works element-wise because NumPy is vectorized.
y = x**2 + 2*x + 1

print(y)

Output: array of transformed values (element-wise).

Time: O(n) • Space: O(n).

15) Debugging Practice

Case 1: shape mismatch

bug
broadcast fails
A.shape = (2,3)
B.shape = (4,)
A + B  # error

Fix: check A.shape and reshape B to be compatible.

Case 2: dtype integer division surprises

bug
integer dtype
x = np.array([1,2,3], dtype=int)
print(x / 2)  # may produce floats, but check dtype rules

Fix: cast to float with x.astype(float) before division (concept).

Case 3: slicing off by one

bug
end exclusive
a[1:3]  # includes 1 and 2, not 3

Fix: remember Python slices are end exclusive.

16) Common Errors

  • ValueError: broadcasting shape mismatch.
  • IndexError: wrong indices for array dimensions.
  • TypeError: operations with non-numeric types (concept).
  • OverflowError/overflow wrap: arithmetic overflow in integer dtypes (concept).

17) Best Practices

  • Use vectorized operations instead of Python loops.
  • Check shape and dtype when results are strange.
  • Use masks for filtering instead of manual loops (concept).
  • Cast dtype when needed (e.g., to float).
  • Prefer stable numerical operations for large values (concept).

22) Answer Key (Explain every answer)

Key explanations

  • NumPy: library for fast numeric arrays and vectorized operations.
  • Array: grid of elements with a specific dtype.
  • Vectorization: perform operations on entire arrays at once (less Python looping).
  • shape: shows dimensions like (rows, cols).
  • slicing: selects a range; end is exclusive.
  • broadcasting: lets scalars/smaller arrays combine safely with larger arrays.
  • sum/mean: compute statistics across array elements.
  • check shape: broadcasting errors happen when shapes are incompatible.
