1) Introduction

Numbers are hard to understand quickly. Charts make patterns visible.

In this chapter, you learn the basics of data visualization.

2) Theory

2.1 Chart types and use cases

  • Line chart: trends over time.
  • Bar chart: categories and comparisons.
  • Scatter plot: relationship between two numeric variables.

2.2 Axes and scale

X-axis is usually “independent variable”. Y-axis shows values. The scale must be honest.

⚠ Warning

Don’t distort charts by cutting off the axis without explanation. Misleading charts harm trust.

3) Syntax

Python
Matplotlib line plot (concept)
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 8, 6, 4]

plt.plot(x, y, marker="o")
plt.title("Trend Example")
plt.xlabel("X values")
plt.ylabel("Y values")
plt.grid(True)

plt.show()

Time: O(n) for plotting points • Space: O(n).

Alternative: use libraries like Seaborn for higher-level plots.

4.3 Advanced Example — labeling + annotation (concept)

Python
add a highlighted point
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [2, 3, 5, 4]

plt.plot(x, y, marker="o")

# highlight a point (x=3, y=5)
plt.annotate(
    "peak",
    xy=(3, 5),
    xytext=(3.2, 5.6),
    arrowprops={"arrowstyle": "->"}
)

plt.title("Trend with Annotation")
plt.xlabel("X")
plt.ylabel("Y")

plt.show()

15) Debugging Practice

Case 1: x and y lengths mismatch

bug
length mismatch
x=[1,2,3]
y=[10,20]
plt.plot(x, y)

Fix: check lengths and filter data accordingly.

Case 2: missing labels

bug
order issue
plt.show()
plt.xlabel("X")

Fix: set title/labels before plt.show().

Case 3: chart looks blank

bug
empty data
y=[]
plt.plot(x,y)

Fix: verify data not empty and print sample values.

16) Common Errors

  • ValueError: x and y lengths mismatch.
  • TypeError: non-numeric data in numeric plot.
  • FileNotFoundError: saving to an invalid path (concept).
  • ImportError: Matplotlib not installed.

17) Best Practices

  • Always add title and axis labels.
  • Choose chart type based on question.
  • Use readable font sizes and grid lines.
  • Avoid misleading axis scaling.
  • Annotate key points (max/min/outliers) when helpful.

22) Answer Key (Explain every answer)

Key explanations

  • Visualization: helps humans see patterns quickly.
  • Line chart: shows trend across ordered values (often time).
  • Bar chart: compares categories by height.
  • Scatter: shows how two numeric variables relate.
  • Good chart: title, axis labels, and clear scales.
  • Honest scaling: prevents wrong conclusions.
  • Annotation: highlights important points or insights.
